Resort Booking Application (MERN Stack)

A simple resort/hotel booking application built using React.js, Redux Toolkit, Node.js, Express.js, and MongoDB.
Users can submit booking requests from the website, and admins can view all bookings from an admin dashboard.

 Features
Beautiful homepage with hero banner, services, and gallery

Booking dialog using React Hook Form + MUI

Form validation with Yup

Booking POST request using Redux Toolkit + Axios

Thank-you message after booking

 Admin Features

Admin dashboard page

Fetch all bookings using Redux

Display bookings in a Material UI table

Auto-formatted dates

 Backend API Features

CRUD API for bookings

Input validation & sanitization

MongoDB database integration

Error handling & clean controller structure

 Tech Stack
